## Commit Convention

| Icon |  Prefix  | Description      |
| :--: | :------: | ---------------- |
|  ✨  |   Feat   | 기능 추가        |
|  🐛  |   Fix    | 버그 수정        |
|  ♻️  | Refactor | 리팩토링         |
|  💄  |  Design  | 스타일 변경      |
|  🎉  |  Chore   | 설정 변경        |
|  🚚  |  Rename  | 폴더/파일명 변경 |
|  🔥  |  Remove  | 파일 삭제        |
|  📝  |   Docs   | 문서 추가/수정   |
